1 - O que são operações de “Commit”, “Push” e “Pop” efetuadas num repositório?
o comando "commit" serve para guardar as alterações dos ficheiros antes de serem enviados para o repositório.
o comando "push" serve para enviar os ficheiros alterados e referenciados por um commit do repositório local para o repositório remoto.
o comando "pop" serve para enviar os ficheiros alterados para uma área temporária que pode ser eliminada a posterior.


2 – Quais sáo as vantagens de usar Branchs num repositório git?
As branches vão permitir realizar desenvolvimentos ou alterações em ficheiros sem eliminar modificações por parte de outros programadores. Funcionam como separadores.
